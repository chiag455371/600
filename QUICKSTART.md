# 快速開始指南 | Quick Start Guide

## 🚀 立即開始使用

### 第一步: 訪問網站

1. 開啟首頁: **index.html**
2. 選擇一個語言進入商品頁面
3. 體驗完整的購物流程

---

## 🔑 管理員首次設定 (必讀!)

### 1. 登入後台

- 訪問: `/admin/`
- 帳號: `admin`
- 密碼: `admin123`

### 2. 設定 LINE 官方帳號 (重要!)

1. 進入「**系統設定**」
2. 填寫你的 LINE 官方帳號連結
3. 格式: `https://line.me/R/ti/p/@yourlineid`
4. 點擊「儲存 LINE 設定」

💡 **如何取得 LINE 連結?**
- 進入 LINE 官方帳號管理後台
- 找到「加入好友」連結
- 複製完整網址

### 3. 新增商品分類 (可選)

系統已預設 5 個分類,如需新增:

1. 「系統設定」→「商品分類管理」
2. 點擊「新增分類」
3. 填寫各語言名稱 (至少填中文)
4. 儲存

### 4. 新增商品

1. 進入「**商品管理**」
2. 點擊「新增商品」
3. 填寫必填項目:
   - 選擇分類
   - 商品名稱 (中文)
   - 商品圖片 URL (至少一張)
4. 儲存

💡 **圖片來源建議**:
- 使用 Unsplash 免費圖片: `https://unsplash.com/`
- 或上傳到圖床後取得 URL
- 格式: `https://example.com/image.jpg`

### 5. 新增聯盟成員 (推廣者)

1. 進入「**聯盟成員**」
2. 點擊「新增成員」
3. 填寫資料:
   - 成員名稱 (必填)
   - 所屬站點/宿舍
   - 單件傭金 (預設 NT$150)
   - 聯絡方式
4. 儲存

### 6. 生成專屬推廣連結

1. 在聯盟成員列表
2. 點擊「QR Code」圖示
3. 下載 QR Code 或複製連結
4. 分享給聯盟成員

## 📱 客戶使用流程

### 情境 1: 一般客戶

1. 訪問首頁
2. 選擇語言 (🇻🇳🇹🇭🇵🇭🇮🇩)
3. 瀏覽商品
4. 點擊「訂購」按鈕
5. 填寫表單
6. 送出後跳轉 LINE

### 情境 2: 透過聯盟成員

1. 掃描聯盟 QR Code
2. 系統自動記錄來源
3. 其餘流程同上
4. **差異**: 訂單自動綁定聯盟成員

## 📊 後台管理日常操作

### 查看新訂單

1. 進入「**儀表板**」
2. 查看「新訂單」數量
3. 或進入「**訂單管理**」查看詳情

### 處理訂單

1. 進入「**訂單管理**」
2. 找到訂單,點擊「編輯」圖示
3. 更新狀態:
   - 新訂單 → 已聯絡
   - 已聯絡 → 已出貨
   - 已出貨 → 已完成
4. 可填寫備註
5. 儲存

### 聯絡客戶

1. 訂單中有客戶的 LINE ID
2. 點擊 LINE ID 連結
3. 直接跳轉到 LINE 聊天

### 匯出報表

1. 進入「**訂單管理**」
2. 點擊右上角「匯出 CSV」
3. 自動下載 Excel 可開啟的 CSV 檔案

### 查看聯盟業績

1. 進入「**聯盟成員**」
2. 查看每位成員的:
   - 導入訂單數
   - 總傭金金額
3. 每月結算時參考此數據

## ⚠️ 常見問題

### Q1: 圖片顯示不出來?

**A**: 檢查圖片 URL 是否正確
- 必須是公開可訪問的網址
- 建議使用 HTTPS 開頭
- 可使用 Unsplash、Imgur 等圖床

### Q2: 多語言圖片怎麼管理?

**A**: 採用 Fallback 機制
- **初期**: 只上傳中文圖片即可 (所有語言共用)
- **進階**: 逐步補充各語言版本
- **顯示**: 當前語言沒有圖片會自動顯示其他語言

### Q3: 聯盟連結有效期多久?

**A**: 30 天
- 客戶點擊聯盟連結後,Cookie 保存 30 天
- 30 天內下單都會綁定到該聯盟成員

### Q4: 如何修改管理員密碼?

**A**: 目前版本為簡易認證
- 需修改 `/js/admin-auth.js` 檔案
- 找到 `ADMIN_CREDENTIALS` 物件
- 修改 `password` 值
- **未來版本**: 會加入後台修改功能

### Q5: 訂單可以刪除嗎?

**A**: 目前版本為「軟刪除」
- 無法直接刪除訂單
- 可將狀態改為「已取消」
- 保留歷史紀錄以便追蹤

### Q6: 支援哪些瀏覽器?

**A**: 現代瀏覽器皆支援
- ✅ Chrome / Edge (建議)
- ✅ Safari
- ✅ Firefox
- ✅ 手機瀏覽器
